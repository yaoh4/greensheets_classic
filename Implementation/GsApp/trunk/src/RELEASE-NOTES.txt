INTRODUCTION
============
This document contains the release notes for this version of the GREENSHEETS application
It is primarily a intended for downstream developers with the  main emphasis on
maintenance and code cleanup releases, with occasional new features.


NEW FEAURES
===========
#4016  Store user preferences for Program role users and use in grants query. Current release provides for
       grants queries using user entered values as well as restoring application default preferences. Future 
       release will provide for manipulating individual user preferences.
