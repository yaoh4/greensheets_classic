INTRODUCTION
============
This document contains the release notes for this version of the GREENSHEETS application
It is primarily a intended for downstream developers with the  main emphasis on
maintenance and code cleanup releases, with occasional new features.


NEW FEAURES
===========
#4016  Provide user customizable preferences for Program role users which are used to query the
       grant collection. The system also contains a default set of query prefences which are used 
       when no user preferences exist for a particular user.
